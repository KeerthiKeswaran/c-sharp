# C# .NET Tasks

This repository contains a collection of C# .NET console application tasks and practice projects, designed to learn and strengthen skills in object-oriented programming, data structures, and core .NET concepts.

## Contents
- Multiple task folders, each with a focused programming exercise
- Example solutions and code for each task
- Step-by-step learning path in the `docs/` folder

## How to Use
- Browse each task folder for individual exercises and solutions
- Follow the instructions in each task's `README.md` for setup and usage
- Use the `docs/learning-path.md` for a guided learning experience

## Tech Stack
- C#
- .NET 10.0
